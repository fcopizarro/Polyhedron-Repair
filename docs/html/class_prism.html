Detailed Description

Representa un prisma, una figura poliedral con 6 vértices, 9 aristas y 5 caras.

La clase Prism hereda de Polyhedron e implementa métodos específicos para el cálculo de métricas y simulación de movimientos asociados a un prisma.

void Prism::CalculateAR ( )
overridevirtual

Calcula el aspecto de relación (AR) del prisma.

AR se calcula como la relación entre la distancia mínima y máxima entre los vértices del prisma. El método primero calcula todas las distancias entre pares de vértices y luego determina la distancia mínima y máxima. Finalmente, AR se calcula como: AR = min_dist / max_dist.

◆ CalculateJs()

void Prism::CalculateJs ( )
overridevirtual

Calcula el valor de Js para cada vértice del prisma.

Js se calcula para cada uno de los 6 vértices del prisma usando la fórmula: Js = dot(normalized1, cross(normalized2, normalized3)), donde normalized1, normalized2 y normalized3 son vectores normalizados entre el vértice actual y los vértices adyacentes.

Reimplemented from Polyhedron.

◆ GetAdjs()

std::tuple< int, int, int > Prism::GetAdjs ( int index)
overridevirtual

Obtiene los índices de los vértices adyacentes a un vértice específico en un prisma.

Dado un índice de vértice dentro de un prisma, esta función devuelve una tupla con los índices de los tres vértices adyacentes.

Parameters
indexEl índice del vértice para el cual se desean obtener los vértices adyacentes. Debe estar en el rango [0, 5] para un prisma.
Returns
Una std::tuple<int, int, int> que contiene los índices de los tres vértices adyacentes al vértice dado. Si el índice proporcionado no está en el rango válido, la función devuelve {0, 0, 0} como valor predeterminado.

Implements Polyhedron.
