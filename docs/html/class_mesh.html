Detailed Description

Representa una malla 3D.

La clase Mesh encapsula los datos y métodos necesarios para manejar y renderizar una malla 3D, incluyendo sus vértices, caras y caja de contorno. También proporciona funcionalidad para dibujar la malla y sus normales, actualizar las posiciones de los vértices y gestionar los datos poliedrales asociados.
