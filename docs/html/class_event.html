Detailed Description

Clase que maneja los eventos de entrada y actualización.

La clase Event es responsable de gestionar los eventos de entrada, como la interacción del usuario con la interfaz gráfica y los eventos del ratón. También se encarga de procesar estos eventos para actualizar la escena y la interfaz de usuario según sea necesario.

Maneja los eventos de entrada.

Procesa los eventos de entrada de SDL, incluyendo el cierre de la aplicación y las interacciones del mouse. También gestiona los eventos de la interfaz de usuario (UI) usando ImGui.
